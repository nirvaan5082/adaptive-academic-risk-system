# Adaptive Academic Risk & Performance Intelligence System

## Overview
This project is an AI-driven academic risk assessment system built using machine learning and predictive analytics. The system evaluates student academic indicators and classifies students into risk categories while generating targeted intervention strategies.

## Features
- Student performance risk prediction
- Risk calibration system
- High / Moderate / Low risk classification
- Personalized recommendation engine
- Feature importance analysis
- Real-time user input support

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Methodology
- Data preprocessing
- Feature engineering
- Classification modeling
- Risk segmentation
- Predictive analytics

## Key Insights
- Overall academic consistency strongly influences student outcomes
- Reading and writing performance significantly impact academic success
- Structured performance categorization improves interpretability
